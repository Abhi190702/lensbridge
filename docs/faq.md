# FAQ

## Does V1 create a webcam device?

No. V1 creates a desktop preview. Virtual camera output is V2.

## Does video leave my network?

Not by default. V1 has no cloud relay.

## Does LensBridge use Bluetooth for video?

No. Bluetooth is only a future pairing/signaling idea.

## Why does phone camera access fail over HTTP?

Browsers require secure contexts for camera APIs. Local HTTPS/WSS is planned.
